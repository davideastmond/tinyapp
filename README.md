# Tiny App Project

## About

A web URL-shortening service. Create short URL links of your favorite sites.

## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session

## Getting started

- Install all dependencies using the npm install command.
- Run the development web server using the node express_server.js command.
